File Referensi

Memperbarui File Referensi

Semua file referensi diperbarui dalam Tampilan Dinamis Smartsheet. File referensi berfungsi sebagai input untuk daftar dropdown dalam formulir KoBoToolbox dan berisi data yang ditambahkan ke kumpulan data master melalui gabungan. Mereka berisi informasi lengkap tentang tingkat administrasi komunitas nelayan (negara, provinsi, kabupaten, desa), lokasi pendaratan, tempat penangkapan ikan, nelayan, pembeli, pengumpul data, spesies, area pengelolaan, jenis peralatan, dan organisasi mitra. Langkah-langkah di bawah ini menguraikan cara memperbarui file referensi dengan informasi baru.

Langkah 1: Buat akun smartsheet gratis.

  • Akun gratis dapat dibuat di https://www.smartsheet.com/

  • Pilih “Coba smartsheet gratis”

Langkah 2: Berikan Barefoot Ocean alamat email yang Anda gunakan untuk membuat akun smartsheet Anda.

  • Barefoot Ocean kemudian akan membagikan file referensi yang terkait dengan organisasi Anda.

  • Untuk melihat file referensi Anda, Anda harus masuk ke Tampilan Dinamis smartsheet menggunakan kredensial smartsheet Anda: https://dynamicview.smartsheet.com/login

  • Setelah masuk ke Tampilan Dinamis (bukan aplikasi smartsheet), Anda akan melihat daftar file referensi yang dibagikan dengan akun Anda (Gambar 1).

Gambar 1: Daftar file referensi dalam Tampilan Dinamis

Langkah 3: Klik pada file referensi untuk membuat pembaruan

  • Setelah terbuka, Anda dapat mengeklik setiap baris untuk mengedit data yang ada, atau mengeklik tombol “Baru” di sudut kanan atas untuk menambahkan data baru (Gambar 2).

  • Panel detail akan muncul di sisi kanan tempat Anda akan memasukkan informasi yang diperlukan (Gambar 3; Tabel 1). Pilih “simpan” di kanan bawah panel detail setelah Anda selesai memperbarui. Anda tidak perlu memberi tahu Barefoot Ocean saat Anda membuat perubahan atau penambahan pada lembar referensi. Ini akan secara otomatis digabungkan dengan file referensi master dan digabungkan dalam Kobo.

  • Untuk memberikan komentar atau mengajukan pertanyaan apa pun kepada tim Barefoot, pilih tab “Komentar” di panel detail. Semua individu yang berbagi file referensi, termasuk tim Barefoot Ocean, akan menerima pemberitahuan saat komentar dibuat, jadi tidak perlu menandai individu tertentu.

Langkah 4: Menanggapi permintaan pembaruan saat data hilang atau salah, atau menanggapi komentar.

  • Jika ada informasi yang tidak lengkap atau salah yang penting untuk mengisi dropdown Kobo atau untuk gabungan, atau jika komentar dibuat di lembar, permintaan pembaruan akan dikirim ke semua pengguna yang sesuai, meminta mereka untuk memperbarui informasi atau menanggapi komentar (Gambar 4). Secara default, semua orang yang memiliki akses ke file referensi Anda akan menerima permintaan pembaruan.

  • Pratinjau data yang perlu diperbarui atau ditanggapi akan muncul di bawah pesan email, tetapi Anda tidak akan dapat memperbarui data apa pun secara langsung di email

    • Di bagian bawah email, ada tautan untuk “Buka lembar”; namun, lembar yang mendasarinya memiliki akses terbatas. Harap tinjau data Anda atau tanggapi komentar tersebut melalui permintaan terbuka.

  • Klik tombol “Buka Permintaan” di pesan email.

  • Jendela browser internet baru akan terbuka dengan pengaturan seperti formulir untuk setiap entri (Gambar 5)

  • Tinjau bidang BFO dan isi bidang yang dapat diedit.

  • Klik berikutnya di bagian bawah layar untuk menelusuri setiap entri yang perlu diperbarui.

  • Untuk melewati entri (mis. jika Anda belum yakin jawaban mana yang harus dipilih) klik “Berikutnya” tanpa membuat pilihan di kolom yang dapat diedit. Entri berikutnya yang perlu diperbarui akan muncul.

  • Jika Anda perlu keluar dari halaman atau berhenti sebelum semua entri diperbarui, pilihan Anda akan disimpan saat berikutnya Anda membuka kembali permintaan. Saat membuka untuk melanjutkan, cukup klik “Next” hingga Anda menemukan entri yang perlu diperbarui.

  • Saat Anda masuk ke entri terakhir, klik “Selesai”. Pesan pop-up akan muncul menanyakan apakah Anda ’ Siap mengirimkan pembaruan Anda?’:

    • Klik “Kembali” jika Anda perlu meninjau

    • Klik “Kirim Pembaruan” untuk mengirimkan pembaruan Anda.

    • Pembaruan Anda secara otomatis ditambahkan ke file referensi dan Anda dapat melihatnya dalam Tampilan Dinamis

Jika sewaktu-waktu Anda menemukan kesalahan, memiliki pertanyaan, atau memerlukan dukungan untuk memperbarui file referensi, silakan hubungi kami di katie@barefootocean.org atau brittany@barefootocean.org.

Gambar 4: Contoh email “Permintaan Pembaruan” saat informasi perlu diperbaiki atau komentar perlu ditanggapi.
